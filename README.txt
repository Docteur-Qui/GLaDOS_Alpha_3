# GLaDOS_Alpha_3
il s'agit de mon premier OS trais basique
mettre ce dossier dans "program file (x86)"
commande:
'add user' rajouter un utilisateur
'clear' néttoi lécran
'reboot' redemaré le programe
'exit' quiter le programe
'new' créer un fichier (il faut preciser le nom appré avoir fais entré)
'edit' éditer un fichier (il faut preciser le nom appré avoir fais entré)
'execute' démarer un programe (il faut preciser le nom appré avoir fais entré)
'rm' suprime un fichier (il faut preciser le nom appré avoir fais entré)
'remove user' suprime des utilisateur (il faut preciser le nom appré avoir fais entré)

merci de l'avoir télecharger
et dite moi quoi faire pour l'améliorer
